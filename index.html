Iivari is a digital signage system

Express

Produces image slideshow from local filesystem.
Tailored to complement raspbian form factor.
See iivari-express in the wiki.

Server

Slide management interface for organisations.
See the server installation instructions.

Client

Puts your slides on display where Qt, WebKit and PySide are provided.
See the client installation instructions.
