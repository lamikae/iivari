Welcome to Iivari

Iivari is a digital signage system.
Should you need a simple image slideshow for leisure, look at the express setup.
If your interests are forking in organisational context, mind the server codebase.
Client teams up with WebKit, Qt, and Python to operate.

Client

Qt WebKit application.
See the client installation instructions in the wiki

Server

Provides slide management interface for organisations.
Ruby on Rails application.
See the server installation instructions in the wiki

Express

Simple image signage system. Codename `raspbian`.
Node.js application.
See iivari-express in the wiki.

Express packages for Debian and Ubuntu variants
