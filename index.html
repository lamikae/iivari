Iivari is a digital signage system.

Express

Simple image feeder. Tailored for single user raspbian device.
See iivari-express in the wiki.

Server

Provides slide management interface for organisations.
See the server installation instructions.

Client

Shows your screens on display should Qt, WebKit and PySide be provided.
