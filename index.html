Welcome to Iivari

Iivari is a digital signage system. The Rails backend server with LDAP integration is suitable for larger organisations. The node.js iivari-express backend might be a better fit in small embedded systems. The same client can operate with both of them.

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

Iivari Express

Node.js backend for iivari-client to serve local images.

See the installation instructions in the wiki

Iivari client

Provides a custom stand-alone WebKit display.
